# GroupDNA — Your WhatsApp Group Chat, Decoded

> *"Spotify Wrapped, but for your friend group."*

A Python + NumPy analytics tool that reads a raw WhatsApp chat export and produces a beautiful, formatted personality report — who's the spammer, who's the night owl, who's the ghost, and what words your group is obsessed with.

Built using **only Python fundamentals and NumPy**. No pandas. No matplotlib. No regex. Just loops, dicts, strings, and one NumPy matrix.

##  What It Analyses

The notebook produces **8 analytical sections**:

1. **Chat Parser** — Reads the raw `.txt` export, handles all edge cases (system messages, media, deleted messages, multi-line messages)
2. **Group Overview** — Total messages, date range, per-person message counts with visual bars
3. **Activity Peaks** — Busiest day and busiest hour across the entire chat period
4. **Activity Heatmap** — A 6×24 NumPy matrix (person × hour) rendered as a text-art heatmap using `░ ▒ █` block characters
5. **Top Words** — Group-wide word frequency with stop-word filtering and bar chart output
6. **Response Patterns** — Average response time per person, fastest and slowest repliers
7. **Silent Streaks** — Longest consecutive days with zero messages per person
8. **Personality Archetypes** — 9 archetypes assigned exclusively based on quantitative rules

---

## The 9 Personality Archetypes

| Archetype | Detection Rule |
|---|---|
| THE SPAMMER | Highest avg consecutive message burst |
| THE GROUP MOM | Highest count of caring keywords |
| THE NIGHT OWL | 60%+ messages between 23:00–04:59 |
| THE STORYTELLER | Avg words per message > 30 |
| THE DRAMA QUEEN | 30%+ messages ALL-CAPS or 2+ exclamation marks |
| THE GHOST | Silent on 60%+ of days in the chat period |
| THE COMEDIAN | Highest % of messages with lol/lmao/haha/rofl |
| THE QUESTION MASTER | 25%+ messages ending with '?' |
| THE LATE NIGHT PHILOSOPHER *(bonus)* | Deep/existential words used after midnight |

Archetypes are assigned **exclusively** — if two people qualify, only the higher scorer gets the label.

## What I Used /  What I Forbade

| Allowed | Forbidden |
|---|---|
| Python fundamentals (loops, dicts, lists, sets, tuples, functions, f-strings) | `pandas` — no DataFrame, no read_csv |
| `numpy` (np.zeros, indexing, slicing, sum, argmax, max) | `matplotlib / seaborn / plotly` — visualization is text-based |
| `open()` and file reading | `collections.Counter` — built own counter with dict |
| `datetime` (strptime, timedelta only) | `collections.defaultdict` — used `.get()` instead |
| All string methods (split, strip, lower, isupper, count, endswith) | `re` (regex) — used string methods for all parsing |
| `sorted()` with key= and reverse= | Any pre-built WhatsApp analyzer library |
| List/dict comprehensions | Any ML/NLP library (scikit-learn, nltk, etc.) |

##  How to Run

### Option 1 — Google Colab (Recommended)

1. Open [colab.research.google.com](https://colab.research.google.com)
2. Upload `GroupDNA_<YourName>_<RollNo>.ipynb`
3. In the left sidebar, click the **Files** icon → upload `hostel_bois.txt`
4. Runtime → **Run All**

The file will be available at `/content/hostel_bois.txt` automatically.

### Option 2 — Jupyter Notebook (Local)

```bash
# 1. Clone the repo
git clone https://github.com/<your-username>/GroupDNA.git
cd GroupDNA

# 2. Install the only dependency
pip install numpy

# 3. Launch Jupyter
jupyter notebook

# 4. Open the .ipynb file and run all cells top to bottom
```

**Requirements:** Python 3.8+, NumPy. That's it.
